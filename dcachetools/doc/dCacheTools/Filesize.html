NAME

dCacheTools::Pool - Find the size of a file in all possible ways known to dCache

SYNOPSIS

  use dCacheTools::Filesize;
  my $fileid = shift; # must be a valid pnfsid/pfn
  my $obj    = dCacheTools::Filesize->new({ fileid => $fileid });
  $obj->show;

REQUIRES

  BaseTools::MyStore
  dCacheTools::PnfsManager

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

dCacheTools::Filesize is a class that holds all the information about the size of a pnfsid/pfn. This includes the size stored in the PnfsManager, the various pool replica size and the one known to pnfs.

Public methods

  • new ($attr): object reference

    Class constructor. Requires a hash reference that must contain a fileid parameter.

  • pnfssize (None): $size

    Returns filesize known to pnfs

  • storedsize (None): $size

    Returns filesize known to the PnfsManager

  • replicas (None): $replicas

    Returns a hash reference that contains the pool replica size.

      $replicas = {
              'cmsdcache2_4' => 24953011,
              'cmsdcache9_6' => 24953011
            };
  • show (None): None

    Formats and dumps all the above information in a single line.
