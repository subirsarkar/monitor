NAME

dCacheTools::PoolGroup - Collects information about a Pool Group

SYNOPSIS

  use dCacheTools::PoolGroup;
  my $obj = dCacheTools::PoolGroup->new({ name => 'cms'});
  $obj->show;

REQUIRES

  Carp
  Data::Dumper
  dCacheTools::PoolManager
  dCacheTools::Pool

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

dCacheTools::PoolGroup collect information about a Pool Group. It is built upon dCacheTools::PoolManager and dCacheTools::Pool. The current implementation is incomplete in the sense that it only holds the space information but nothing about the links, ugroup etc.

Public methods

  • new ($attr): object reference

    Class constructor.

        $attr->{name} - Pool Group name
  • info (None): $info

    Returns the underlying container which is a hash reference

  • show (None): None

    Displays the information for the Pool Group a la poolinfo (style).
