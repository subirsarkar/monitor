NAME

dCacheTools::TransferStatus - Summarises status of each type of transfer

SYNOPSIS

  use dCacheTools::TransferStaaus;
  my $transferType = shift || q|dcap-cmsdcdcap|;
  my $obj = dCacheTools::TransferStatus->new({ type => $transferType });
  $obj->show;

REQUIRES

  BaseTools::MyStore
  dCacheTools::PoolManager
  dCacheTools::Pool
  dCacheTools::PnfsManager

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

dCacheTool::TransferStatus loops of over all the pools, executes mover ls and collects information for a type of transfer, namely dcap, gridftp, RemoteGsiftpTransfer etc. The information collected here should be combined with that found in individual doors in case we want to prepare a comprehensive report similar to what is found on the dCache monitor.
