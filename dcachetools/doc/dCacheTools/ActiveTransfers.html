NAME

dCacheTools::ActiveTransfers - Parses the Active Transfer table on the dCache web monitor

SYNOPSIS

  use dCacheTools::ActiveTransfers;
  my $obj = dCacheTools::ActiveTransfers->new({ webserver => 'cmsdcache'});
  $obj->show;

REQUIRES

  Carp
  WebTools::Page

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

dCacheTools::ActiveTransfers parses the dCache Active Transfers table embedded in http://<webserver>:2288/context/transfers.html and stores in a data structure that can be easily used by other tools. This information can be used to calculate transfer rates for dcap, gridftp etc. for individual pools which serves as a very useful diagnostic tool. This is also a very efficient way to access such information as the Admin Console is not used.

Public methods

  • new ($attr): object reference

    Class constructor.

      $attr->{webserver} - dCache web server host
  • rows (None): $rows

    Return the underlying container which is a hash reference

  • show (None): None

    Display the information.

      print Data::Dumper->Dump([$rows], [qw/rows/]);
