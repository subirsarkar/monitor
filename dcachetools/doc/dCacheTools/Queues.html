NAME

dCacheTools::Queues - Parses the dCache Pool Mover Queues table

SYNOPSIS

  use dCacheTools::Queues;
  my $obj = dCacheTools::Queues->new({ webserver => q|cmsdcache| });
  $obj->show;
  print $obj->movers({ pool => q|cmsdcache8_1|, type => q|default|, state => q|Active|}), "\n";

REQUIRES

  WebTools::Page

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

dCacheTools::Queues parses the dCache Pool Mover Queues table embedded in http://webserver:2288/queueInfo and stores in a data structure that can be easily used by other tools. The same information can be extracted from the PoolManager cell also parsing the cm ls -r output.

Public methods

  • new ($attr): object reference

    Class constructor.

        $attr->{webserver}  - dCache web server host
  • show (None): None

    Display the information.

  • movers ($params): scalar

    Returns the number of movers assigned to a particular type and state of transfer. The types are Movers, Restores, Stores, P2P-Server, P2P-Client etc. while the states are Active, Max and Queued

       $params->{pool}  - Pool name
   $params->{type}  - transfer/mover type
   $params->{state} - state of the transfer
