NAME

dCacheTools::Mover - Parses the Pool mover ls output for a single pnfsid

SYNOPSIS

  use dCacheTools::Mover;
  my $input = q|11561 A H {GFTP-cmsdcache7-Unknown-109@gridftp-cmsdcache7Domain:10007} 000800000000000001F79888 h={SU=1682243584;SA=1730150400;S=None} bytes=1680146432 time/sec=6224 LM=0|;
  my $obj = dCacheTools::Mover->new($input);
  $obj->show;

REQUIRES

  Data::Dumper
  BaseTools::Util

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

dCacheTools::Mover is there for convenience. It parses the Pool mover ls command output and provides setter methods to get individual fields. All the accesssor are AUTOLOADed.
