NAME

dCacheTools::Space - Parses the dCache Usage Info tables available on the dCache monitor, either for the whole system or for a particular Pool group.

SYNOPSIS

  use dCacheTools::Space;
  my $obj = dCacheTools::Space->new({ webserver => 'cmsdcache', pgroup => 'cms'});
  $obj->show;
  # reset to global
  $obj->reset;
  $obj->show;

REQUIRES

  Carp
  WebTools::Page

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

dCacheTools::Space extracts and parses the dCache Usage Info table embedded in http://<webserver>:2288/usageInfo or in http://<webserver>:2288/pools/list/PoolManager/<pgroup>/spaces/ and stores in a data structure that can be easily used by other tools.

Public methods

  • new ($attr): object reference

    Class constructor.

        $attr->{webserver} - dCache web server host
    $attr->{pgroup}    - Pool Group name
  • show (None): None

    Display the information as a formatted dump,

  • webserver ($webserver): $webserver

    Setter/getter of $webserver.

  • pgroup ($pgroup): $pgroup

    Setter/getter of $pgroup.

      my $obj = dCacheTools::Space->new({ webserver => 'cmsdcache' });
  $obj->pgroup('cms');
  $obj->show;
  • rows (None): $rows

    Returns the array reference that holds the table
