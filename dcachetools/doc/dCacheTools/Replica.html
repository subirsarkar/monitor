NAME

dCacheTools::Replica - Parses the Pool rep ls pnfsid output.

SYNOPSIS

  use dCacheTools::Replica;
  my $input = q|000800000000000001F13420 <--C-------L(0)[1]> 2676334281 si={cms:alldata}|;
  my $obj = dCacheTools::Replica->new($input);
  $obj->show;

REQUIRES

  Data::Dumper
  BaseTools::Util

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

dCacheTools::Replica is there for convenience. It parses the Pool rep ls pnfsid command output and provides getters to get individual fields. All the methods are AUTOLOADed. Consult http://trac.dcache.org/trac.cgi/wiki/manuals/RepLsOutput for a detailed discussion of all the fields.
