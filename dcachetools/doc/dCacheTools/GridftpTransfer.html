NAME

dCacheTools::GridftpTransfer - A wrapper for individual Gridftp transfer door

SYNOPSIS

  my $door = shift || q|GFTP-cmsdcache1-Unknown-2368|;
  my $obj = dCacheTools::GridftpTransfer->new({ name => $door });
  dCacheTools::GridftpTransfer->header();
  $obj->show;

REQUIRES

  BaseTools::ConfigReader;
  BaseTools::Util
  dCacheTools::Cell

INHERITANCE

  dCacheTools::Cell

EXPORTS

none.

DESCRIPTION

dCacheTools::GridftpTransfer is a simple wrapper over an single Gridftp transfer door which parses and combines the door info and get door info command output.
