NAME

BaseTools::FileList - A frontend to File::Find

SYNOPSIS

  use BaseTools::FileList;
  my $list = BaseTools::FileList->new({ path => q|/pnfs/pi.infn.it/data/cms/store/users|, verbose => 1 });
  $list->show;
  $list->path(q|/pnfs/pi.infn.it/data/cms/store/PhEDEx_LoadTest07|);
  $list->show;

REQUIRES

File::Find

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

BaseTools::FileList gathers all the files under a base path recursively using File::Find and fills a hash with filename as key. We use a hash instead of an array in order to support any future development where the value for a key may contains more information about a file.

Public methods

  • new ($attr): object reference

    Class constructor.

        $attr->{path}    - must be specified
    $attr->{verbose} - debug flag
  • show (None): None

    Dump the filelist

  • info ($info): $info

    Get the underlying hash reference whose keys are the filenames

  • files ($strip): $files

    Returns a reference to an array that holds the files found. One can optionally strip the base path, important to compare with the PhEDEx list.

  • traverse (None): None

    Use File::Find to traverse the base path and fill the underlying container with the file list found

  • path ($path): $path

    Set/get base path

  • verbose ($verbose): $verbose

    Set/get verbose mode
