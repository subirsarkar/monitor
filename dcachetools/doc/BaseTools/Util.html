NAME

BaseTools::Util - A collection of utility functions.

SYNOPSIS

  use BaseTools::Util qw/trim readFile/;
  my $content = readFile($infile);
  my @list    = readFile($infile); # wantarray

REQUIRES

  File::Find
  File::Copy
  Net::Domain
  POSIX
  Term::ANSIColor

INHERITANCE

none.

EXPORTS

  trim  
  getTime 
  debug 
  getCommandOutput
  readFile
  getHostname
  message
  writeHTML
  readDir
  traverseDir

DESCRIPTION

A collection of utility functions.

Public methods

  • trim ($str): $str

    Remove leading and trailing spaces of a string and returns the modified string

  • getTime($time): $fmt_timestamp

    Converts a long integer into a formatted string

  • debug ($arg): None

    Prints a debug string by appending $arg

  • getCommandOutput ($command): $content [@content]

    Executes a command and returns the output either as a scalar string or as an array of output lines depending on the context (caller)

  • readFile ($filename): $content [@content]

    Read a file and returns the content either as a single string or as an array of output lines depending on the context (caller)

  • getHostname (None): $hostname

    Executes Net::Domain::hostname and returns the hostname -s

  • message ($color, $text): None

    Print a message string preceded by a time stamp that used Term::ANSIColor

        $color - Color of the time stamp that hints at the status (GREEN, RED, BLUE etc.)
    $text  - The message string
  • writeHTML ($filename, $content): None

    Save the HTML content in a file

        $filename - output file name
    $content  - HTML content to be saved
 
=item * readDir ($dirname): @list

    Finds the files in a directory and returns the list

        $dirname - Directory name
  • traverseDir ($dirname): @list

    Finds the files in a directory recursively and returns the list of files

        $dirname - Directory name
