NAME

BaseTools::ConfigReader - A singleton object that makes it easy for other classes to access the application wide configuration.

SYNOPSIS

  use BaseTools::ConfigReader;
  my $reader = BaseTools::ConfigReader->instance();
  my $admin_node = $reader->{config}{admin}{node};
  # show the content
  $reader->show;

REQUIRES

Data::Dumper

INHERITANCE

none.

EXPORTS

none.

DESCRIPTION

BaseTools::ConfigReader is a singleton class that reads the application wide configuration file which is written in Perl itself and holds the hash reference that contains the configuraiton information. The configuration file should return the hash reference as the last line like in the following example,

  our $cfg =
  {
      admin => {
                 node => qq|cmsdcache|,
              timeout => 300,
                debug => 0,
                delay => 2000,
        discard_error => 1
      }
  };
  $cfg->{PoolManager} =
  {
    activityMarker => 1000
  };
  $cfg;

Public methods

  • instance (None): object reference

    Access the singleton object reference using this static (package level) method. The first time it is accessed in an application the object is created and subsequently the same object is accessed. Since it is supposed to be a read-only object even simultaneous access does not require any special requirement.

  • show (None): None

    Dump the configuration information.
